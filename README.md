# F is for Formatting

A web app that transforms raw resume content (or any structured text) into beautiful formatted resumes and slide decks.

## Features

- **Content Input**: Paste your resume, bio, or project description
- **Formatted Resume**: Clean, professional resume layout with section headers and bullet points
- **Slide Deck**: Present your content as a navigable slide deck—perfect for pitches or presentations

## Formatting Tips

- Use `## Section Name` for section headers (e.g., `## Experience`, `## Education`)
- Use `•` or `-` for bullet points
- The first line is treated as your name/title
- The second line can be a subtitle (e.g., job title, location)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

```bash
npm run build
```

## Tech Stack

- React 18 + TypeScript
